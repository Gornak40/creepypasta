# CreepyPasta

### Описание
* Больше 1000 страшных историй для взрослых и детей
* Удобный консольный интерфейс
* Не нуждается в интернет подключении

### Установка
* ```unzip stories.zip```
* ```sudo -H pip3 install -r requirements.txt```

### Использование
* ```chmod +x main.py```
* ```./main.py```
